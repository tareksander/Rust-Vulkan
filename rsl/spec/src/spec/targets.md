# Backends


This chapter defines the interaction between the language and the compilation target for the backends.


## SPIR-V

This is the primary target with support for all features.




## WGSL

This target cross-compiles to WGSL, but only supports features also possible in WGSL.




## Rust

This target cross-compiles to Rust code for executing the same algorithms on the CPU. It has lower priority than the other targets.
