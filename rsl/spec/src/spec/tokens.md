# Tokens

## Special characters

In general, each special character used in the language (not alphanumeric and not in a string or identifier) gets its own token.

There are some double-character tokens like `::`, and some combinations of special characters, which are listed here:

- `->`: `ThinArrow`
- `=>`: `ThickArrow`




## Identifiers

Identifiers start with an ASCII alphabetic character and continue with a ASCII alphanumeric characters or `_`. When an Identifier with the same content as a keyword is encountered, the keyword takes precedence.

Identifiers are denoted with Ident in the syntax section.


## Keywords

Keywords are denoted by name in the syntax section.
