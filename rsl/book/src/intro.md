# Introduction

<div class="warning">
The documentation is currently a mix of "is now" and "future plans".

It will get more fleshed-out once the language reaches usability with compute shaders, with unfinished sections clearly marked.

This book serves as both a guide to and a specification for the RSL language.

[Chapter 1](./basics.md) explains the basics of the language and ends with a hello-world example, though the concepts should be familiar from Rust.

[Chapter 2](./spec.md) contains the actual language specification.

[Chapter 3](./compiler.md) contains the available compiler flags via command line. For configuration of the compiler library, see the Rust docs.
